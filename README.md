# Image Gallery Project

## Overview
This project is a responsive Image Gallery built using HTML, CSS, and JavaScript. It allows users to view images in a gallery layout and open them in a lightbox with next and previous navigation buttons.

## Features
- Responsive image gallery layout
- Hover effects and smooth transitions
- Lightbox image preview
- Next and Previous navigation
- Mobile-friendly design
- Simple and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

Image_Gallery/
│
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html` in a web browser.
4. Click on any image to open the lightbox view.
5. Use the Next and Previous buttons to navigate through images.
